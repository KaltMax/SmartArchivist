# Batch Processing Documentation

## Overview

SmartArchivist.Batch is a standalone console application for automatically backing up the documents. It exports all document metadata (including OCR text and GenAI summaries) to JSON and downloads all PDF files from MinIO storage, creating timestamped backup archives that can be used for recovery or data migration.

**Key Features:**
- Complete document metadata export (including OCR text and AI-generated summaries)
- PDF file downloads from MinIO storage
- Backup manifest with statistics and error tracking
- Timestamped backup folders for easy versioning
- Configurable via `appsettings.json`

## How to Run

1. Navigate to the SmartArchivist.Batch project directory:
```bash
cd SmartArchivist.Batch
```

2. Build the application:
```bash
dotnet build -c Release
```

3. Run the application:
```bash
dotnet run -c Release
```

## Configuration

The application is configured via the `appsettings.json` file located in the `SmartArchivist.Batch` directory.

### Configuration Settings

| Setting | Description | Default |
|---------|-------------|---------|
| `BackupPath` | Directory where backups are stored (relative or absolute) | `.\\Backups` |
| `ConnectionString` | PostgreSQL connection string | `Host=localhost;Port=5432;Database=smartarchivist;Username=postgres;Password=postgres` |
| `MinIoEndpoint` | MinIO server endpoint | `localhost:9000` |
| `MinIoAccessKey` | MinIO access key | `minioadmin` |
| `MinIoSecretKey` | MinIO secret key | `minioadmin` |
| `MinIoBucketName` | MinIO bucket name where documents are stored | `smartarchivist-documents` |
| `MinIoUseSsl` | Use SSL for MinIO connection | `false` |

## Scheduling with Windows Task Scheduler to run the Backup automatically

1. Open **Task Scheduler**
2. Click **"Create Basic Task"** in the right panel
3. **Name**: "SmartArchivist Daily Backup"
4. **Description**: "Automated backup of SmartArchivist documents and metadata"
5. **Trigger**: e.g, Daily at 1:00 AM
6. **Action**: Start a program
   - **Program/script**: Copy and customize `run-backup-template.bat` with your actual paths
7. **Finish** and review settings

## Backup Structure

Each backup creates a timestamped folder with the following structure:

```
Backups/
└── backup_20251217_124632/
    ├── documents-metadata.json    # Complete document metadata
    ├── backup-manifest.json       # Backup statistics and errors
    └── files/                     # All PDF files
        ├── 3565bbd7-f277-4e07-af8b-fa08c8e1ae30_Document1.pdf
        ├── 7d8e360b-11ab-44ef-9131-b0a14ee2bce5_Document2.pdf
        └── ...
```

### documents-metadata.json

Contains complete document metadata including:
- Document ID, name, file path, extension
- Content type, upload date, file size
- Document processing state
- Full OCR extracted text (OcrText)
- GenAI summary (GenAiSummary)
- Tags generated by GenAI

Example:
```json
[
  {
    "id": "3565bbd7-f277-4e07-af8b-fa08c8e1ae30",
    "name": "Document1",
    "filePath": "3565bbd7-f277-4e07-af8b-fa08c8e1ae30/Document1.pdf",
    "fileExtension": ".pdf",
    "contentType": "application/pdf",
    "uploadDate": "2025-12-14T08:09:48.81554Z",
    "fileSize": 1779898,
    "state": 4,
    "ocrText": "Full extracted text content...",
    "genAiSummary": "AI-generated summary...",
    "tags": ["tag1", "tag2", "tag3"]
  }
]
```

### backup-manifest.json

Contains backup statistics and error information:

```json
{
  "backupDate": "2025-12-17T12:46:32.032906Z",
  "totalDocuments": 6,
  "totalSizeBytes": 5836892,
  "successfulBackups": 6,
  "failedBackups": 0,
  "errors": []
}
```

**Manifest Fields:**
- `backupDate`: Timestamp when backup was created
- `totalDocuments`: Total number of documents in the database
- `totalSizeBytes`: Total size of all backed up files in bytes
- `successfulBackups`: Number of documents successfully backed up
- `failedBackups`: Number of documents that failed to backup
- `errors`: Array of error messages for failed documents

### files/ Directory

Contains all PDF files downloaded from MinIO, named with the pattern:
```
{document-guid}_{original-filename}.pdf
```
